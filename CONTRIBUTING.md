# Contributing to edb-postpic

Merci de votre intérêt pour contribuer.

## Processus
- Ouvrez une issue pour discuter des changements majeurs.
- Créez une branche `feature/xxx` ou `fix/xxx`.
- Soumettez une Pull Request avec une description claire et tests si possible.

## Style
- Respectez le style existant du code.
- Ajoutez des tests pour les changements fonctionnels.
- Mettez à jour `CHANGELOG.md` pour les changements notables.

## Licence des contributions
En contribuant, vous acceptez que votre contribution soit distribuée sous la licence MIT du projet.
